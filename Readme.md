# cinit

A Little More Verbse npm init which will Create better package.json file for
Production Level Packages.

## Usage:

For new existing package.json

```bash
  cd /root/of/pakcage # Where package.json is
  nnpm # that's it
```

Installation:

```bash
npm i -g cinit
```
