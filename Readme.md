# NNPM

A Little More Verbse npm init which will Create better package.json file for
Production Level Packages

Installation:

```bash
npm i -g nnpm
```

## Usage:

For new existing package.json

```bash
  cd /root/of/pakcage # Where package.json is
  nnpm # that's it
```

## What it Checks

1. name:
2. version:
3. description:
4. keywords: []
5. homepage:
6. bugs:
7. license:
8. author:
9. files: []
10. main: | browser
11. bin:
12. repository:
13. config:
